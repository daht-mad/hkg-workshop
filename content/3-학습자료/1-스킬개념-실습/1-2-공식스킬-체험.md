## 실습 1: 스킬 수동 설치하기 (폴더 이해용)

> ⭐ 스킬의 정체가 "폴더"라는 걸 직접 확인해봅시다!

### Step 1: GitHub에서 스킬 다운로드

1. [github.com/anthropics/skills](https://github.com/anthropics/skills) 접속
2. 녹색 **Code** 버튼 클릭 → **Download ZIP** 클릭
3. 압축 해제

![](../../../public/images/CleanShot%202026-02-22%20at%2002.26.17@2x.png)


### Step 2: 우리가 설치할 스킬 2개 찾기

압축 해제한 폴더에서 `skills/` 안을 보면 스킬들이 쭉 나옵니다.

이 중에서 **2개만** 복사할 거예요:

| 스킬 | 하는 일 | 왜 설치하나요? |
|------|--------|--------------|
| **skill-creator** | 스킬을 만들어주는 스킬 | 다음 실습에서 내 스킬 만들 때 사용! |
| **slack-gif-creator** | Slack용 GIF 생성 | 재미있고, 바로 결과가 보여서 체험용으로 딱! |
![](../../../public/images/CleanShot%202026-02-22%20at%2002.27.35@2x%201.png)

```
skills-main/
└── skills/
    ├── ...
    ├── skill-creator/        ← 🎯 이거!
    │   └── SKILL.md
    ├── slack-gif-creator/    ← 🎯 이거!
    │   └── SKILL.md
    └── ...
```

> 💡 **발견**: 스킬 하나 = 폴더 하나 + `SKILL.md` 파일!

### Step 3: 전역 스킬 폴더에 복사

아까 1.1에서 열어봤던 `~/.claude/skills/` 폴더를 다시 열어주세요.

> 💡 기억 안 나면: Finder에서 `Cmd + Shift + G` → `~/.claude/skills` 입력

### Step 4: 스킬 폴더 복사

1. 다운받은 `skills-main/skills/` 안에서 `skill-creator` 폴더와 `slack-gif-creator` 폴더 선택
2. 두 폴더를 **복사** (`Cmd + C`)
3. 아까 찾은 `~/.claude/skills/` 폴더에 **붙여넣기** (`Cmd + V`)

### Step 5: 설치 확인

`~/.claude/skills/` 폴더에 아래 2개가 보이면 성공!

```
~/.claude/skills/
├── skill-creator/        ✅
│   └── SKILL.md
└── slack-gif-creator/    ✅
    └── SKILL.md
```

## 💡 더 쉬운 방법: 명령어 한 줄!

방금 수동으로 한 걸... **명령어 한 줄**로 할 수 있습니다.

### 플러그인이란?

| 개념 | 설명 |
|------|------|
| **플러그인** | 스킬 여러 개를 묶은 **배포 패키지** |
| **마켓플레이스** | 플러그인을 배포하는 **저장소** (GitHub 등) |

> 💡 **비유**: 플러그인 = 앱스토어에서 받는 앱 묶음

### 명령어로 설치하기

> ⚠️ **중요**: `/plugin`, `/skills`, `/mcp` 명령어는 **VSCode 익스텐션에서 안 됩니다!**
> 
> **터미널 환경에서만** 사용 가능합니다.

**Step 0**: 터미널에서 Claude Code 진입
1. **새 터미널** 열기 (Mac: Terminal 앱 또는 VSCode 터미널)
2. 아래 명령어 입력:
   ```bash
   claude
   ```
3. Claude Code가 실행되면 아래 명령어들 사용 가능!

**Step 1**: 마켓플레이스 등록 (최초 1회)
```
/plugin marketplace add anthropics/skills
```

**Step 2**: 스킬 설치
```
/plugin install document-skills@anthropic-agent-skills
```

**Step 3**: 설치 확인
```
/skills
```

> 💡 **결과는 같아요!** 명령어가 GitHub에서 다운로드 → `~/.claude/skills/`에 복사를 자동으로 해줍니다.

### 명령어 해석

```
/plugin install document-skills @ anthropic-agent-skills
              ↑                    ↑
          플러그인 이름         마켓플레이스 이름
```

> 💡 **팁**: Claude.ai 유료 플랜 사용자는 이미 설치되어 있어요!

## 실습 2: 스킬 사용해보기

### Slack GIF 만들기 체험

**Step 1**: Claude Code에서 아래처럼 요청해보세요
```
"환영합니다!" 라는 텍스트가 반짝이는 Slack용 GIF 바탕화면에 만들어줘
```

**Step 2**: 결과 확인
- GIF 파일이 생성됨
- 열어서 애니메이션 확인!
- 줌 채팅창에 서로 올려서 자랑해봐요!

> 💡 이 모든 게 `slack-gif-creator/SKILL.md` 안의 지침대로 동작한 겁니다!

## 스킬이 작동하는 원리

방금 체험한 것을 복기해봅시다:

```
1. 사용자: "Slack용 GIF 만들어줘"
   ↓
2. Claude: (description 확인) "아, slack-gif-creator 스킬이네"
   ↓
3. Claude: (SKILL.md 읽음) 단계별 절차 확인
   ↓
4. Claude: 절차대로 실행 → GIF 생성!
```

**핵심**: `description`에 "Slack", "GIF" 같은 키워드가 있어서 **자동 인식**!

## ⚠️ 문제 해결

### "스킬이 안 보여요"

1. `~/.claude/skills/` 폴더에 `skill-creator`, `slack-gif-creator` 폴더가 있는지 확인
2. 폴더 안에 `SKILL.md` 파일이 있는지 확인
3. Claude Code 재시작

### "스킬이 작동 안 해요"

1. 정확한 키워드 사용 (예: "Slack GIF 만들어줘")
2. Claude Code 재시작 후 다시 시도
3. 강사에게 질문!
