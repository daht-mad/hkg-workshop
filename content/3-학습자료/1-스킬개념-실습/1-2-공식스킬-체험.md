
# 1.2 공식 스킬 체험

> 앤트로픽이 만든 스킬을 직접 설치하고 사용해봅니다.


## 공식 스킬이란?

앤트로픽(Anthropic)에서 직접 만들어 공개한 스킬들입니다.

**어디서 볼 수 있나요?**
- GitHub: [github.com/anthropics/skills](https://github.com/anthropics/skills)

![](../../../public/images/CleanShot%202026-02-22%20at%2002.30.54@2x.png)


## 실습 1: 스킬 수동 설치하기 (폴더 이해용)

> ⭐ 스킬의 정체가 "폴더"라는 걸 직접 확인해봅시다!

### Step 1: GitHub에서 스킬 다운로드

1. [github.com/anthropics/skills](https://github.com/anthropics/skills) 접속
2. 녹색 **Code** 버튼 클릭 → **Download ZIP** 클릭
3. 압축 해제

![](../../../public/images/CleanShot%202026-02-22%20at%2002.26.17@2x.png)


### Step 2: 우리가 설치할 스킬 2개 찾기

압축 해제한 폴더에서 `skills/` 안을 보면 스킬들이 쭉 나옵니다.

이 중에서 **2개만** 복사할 거예요:

| 스킬 | 하는 일 | 왜 설치하나요? |
|------|--------|--------------|
| **skill-creator** | 스킬을 만들어주는 스킬 | 다음 실습에서 내 스킬 만들 때 사용! |
| **slack-gif-creator** | Slack용 GIF 생성 | 재미있고, 바로 결과가 보여서 체험용으로 딱! |
![](../../../public/images/CleanShot%202026-02-22%20at%2002.27.35@2x%201.png)

```
skills-main/
└── skills/
    ├── ...
    ├── skill-creator/        ← 🎯 이거!
    │   └── SKILL.md
    ├── slack-gif-creator/    ← 🎯 이거!
    │   └── SKILL.md
    └── ...
```

> 💡 **발견**: 스킬 하나 = 폴더 하나 + `SKILL.md` 파일!

### Step 3: 전역 스킬 폴더에 복사

아까 1.1에서 열어봤던 `~/.claude/skills/` 폴더를 다시 열어주세요.

> 💡 기억 안 나면: Finder에서 `Cmd + Shift + G` → `~/.claude/skills` 입력

### Step 4: 스킬 폴더 복사

1. 다운받은 `skills-main/skills/` 안에서 `skill-creator` 폴더와 `slack-gif-creator` 폴더 선택
2. 두 폴더를 **복사** (`Cmd + C`)
3. 아까 찾은 `~/.claude/skills/` 폴더에 **붙여넣기** (`Cmd + V`)

### Step 5: 설치 확인

`~/.claude/skills/` 폴더에 아래 2개가 보이면 성공!

```
~/.claude/skills/
├── skill-creator/        ✅
│   └── SKILL.md
└── slack-gif-creator/    ✅
    └── SKILL.md
```


## 실습 2: 스킬 사용해보기

### Slack GIF 만들기 체험

**Step 1**: Claude Code에서 아래처럼 요청해보세요
```
"환영합니다!" 라는 텍스트가 반짝이는 Slack용 GIF 바탕화면에 만들어줘
```

**Step 2**: 결과 확인
- GIF 파일이 생성됨
- 열어서 애니메이션 확인!
- 줌 채팅창에 서로 올려서 자랑해봐요!

> 💡 이 모든 게 `slack-gif-creator/SKILL.md` 안의 지침대로 동작한 겁니다!


## 💡 공식 스킬의 장점

| 장점 | 설명 |
|------|------|
| **검증됨** | 앤트로픽이 직접 테스트 |
| **안정적** | 에러 처리가 잘 되어있음 |
| **참고용** | 내 스킬 만들 때 구조 참고 가능 |


## ⚠️ 문제 해결

### "스킬이 안 보여요"

1. `~/.claude/skills/` 폴더에 `skill-creator`, `slack-gif-creator` 폴더가 있는지 확인
2. 폴더 안에 `SKILL.md` 파일이 있는지 확인
3. Claude Code 재시작

### "스킬이 작동 안 해요"

1. 정확한 키워드 사용 (예: "Slack GIF 만들어줘")
2. Claude Code 재시작 후 다시 시도
3. 강사에게 질문!
