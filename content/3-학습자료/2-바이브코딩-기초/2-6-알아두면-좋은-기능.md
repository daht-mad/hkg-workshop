
# 2.6 알아두면 좋은 기능

> 지금 당장 안 외워도 됩니다. 필요할 때 돌아와서 찾아보세요!


## 📊 /status - 현재 상태 확인

```
/status
```

**보여주는 정보:**
- 현재 **로그인된 계정** (누구로 접속 중인지)
- Claude Code **버전**
- 현재 사용 중인 **AI 모델** (Sonnet, Opus 등)
- 연결 상태

> 💡 **팁**: "내가 어떤 계정으로 로그인했지?" 싶을 때 `/status` 입력!


## 🔄 /resume - 이전 대화 불러오기

```
/resume
```

**언제 쓰나요?**
- 새로 로그인한 후 기존 작업 이어가고 싶을 때
- 터미널을 껐다 켜서 대화가 사라졌을 때
- 어제 하던 작업을 오늘 계속하고 싶을 때

```
예시 흐름:
1. /resume 입력
2. 이전 대화 목록 표시됨
3. 이어가고 싶은 대화 선택
4. 기존 맥락 그대로 작업 계속!
```

> 💡 **팁**: 로그인 새로 하면 대화창이 비어 보이지만, `/resume`으로 기존 대화를 불러올 수 있어요!


## 전체 명령어 한눈에

### 터미널 명령어

| 명령어 | 기능 | 언제 쓰나요? |
|--------|------|-------------|
| `/init` | CLAUDE.md 생성/업데이트 | 프로젝트 시작할 때 |
| `/model` | AI 모델 변경 | 다른 모델 써보고 싶을 때 |
| `/mcp` | MCP 서버 상태 확인 | 연결된 MCP 확인할 때 |
| `/status` | 계정·버전·모델 확인 | 현재 상태 확인할 때 |
| `/login` | 계정 전환 | 다른 계정으로 바꿀 때 |
| `/resume` | 이전 대화 불러오기 | 기존 작업 이어갈 때 |
| `/clear` | 대화 **완전히** 지우기 | 다른 주제 시작할 때 |
| `/compact` | 대화 **요약**해서 줄이기 | 토큰 절약할 때 |
| `/context` | 토큰 사용량 시각화 | 얼마나 썼는지 확인할 때 |
| `/help` | 도움말 보기 | 뭘 할 수 있는지 모를 때 |

### VSCode 확장프로그램 커맨드 메뉴

입력창에 `/` 를 입력하면 커맨드 메뉴가 나타납니다. 터미널과 이름이 다른 것들:

| 메뉴 항목 | 터미널 대응 | 기능 |
|----------|------------|------|
| Switch Model | `/model` | 모델 변경 |
| MCP Status | `/mcp` | MCP 서버 상태 확인 |

> 💡 그 외 `/clear`, `/compact`, `/init` 등은 터미널과 동일하게 입력 가능합니다.
