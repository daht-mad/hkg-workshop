# 클로드코드 바이브코딩 실무 워크샵

> 비개발자도 AI로 업무를 자동화할 수 있습니다. 4시간 동안 직접 만들어봅니다.

---

## 개요

- **일시**: 2026년 3월 17일 (화) 14:00 ~ 18:00
- **시간**: 4시간 (휴식 15분 포함)
- **대상**: PM, 디자이너, 리서처, 마케터 등 비개발 직군
- **목표**: 클로드코드로 본인 업무 자동화 스킬을 만들고, GitHub에 배포하여 팀원과 공유

---

## 전체 흐름

```
[사전 준비]
  클로드코드 설치 + /workshop-prep으로 설계서 생성
      ↓
[1부] 스킬 이해 & 첫 사이클 (1시간)
  개념 → 체험 → 제작 → 테스트 → 배포 → 공유
      ↓
[2부] 바이브코딩 기초 (45분)
  왜 바이브코딩 → CLAUDE.md → MCP → 필수 기능 → Q&A
      ↓
[휴식 15분]
      ↓
[3부] 본인 스킬 만들기 (2시간)
  환경 준비 → 계획 → 생성 → 테스트 → 개선 → 연동 → 완성!
      ↓
[이후] 혼자서도 할 수 있다!
  MCP 연동 추가 + 스킬 개선 + 새 스킬 만들기
```

> 스킬이 뭔지 알고, 직접 만들어보고, 배포까지 경험합니다.

| 세션 | 내용 | 교안 |
|------|------|------|
| **강사 소개** | 강사 소개, 워크샵 목표 안내, 사전 준비 확인 | `2-시작하며` |
| **스킬 개념 & 사례** | 스킬 = 업무 지침서, SKILL.md 구조, 전역/프로젝트 차이 | `1-1-스킬이란` |
| **공식 스킬 체험** | GitHub에서 다운로드 → 수동 설치 → Slack GIF 만들기 체험 | `1-2-공식스킬-체험` |
| **첫 스킬 만들기 & 테스트** | skill-creator로 회의록 요약 스킬 생성 → 테스트 | `1-3-첫-스킬-만들기` |
| **배포 & 공유** | GitHub CLI로 스킬 배포 → 친구 스킬 설치 | `1-4-배포-공유` |

**1부 결과물**: 회의록 요약 스킬 1개 완성 + GitHub 배포
